# 프로젝트 개요

이 문서는 GitHub Actions `lint-and-test` 워크플로우를 통과하기 위한 예시 마크다운 파일입니다.

## 파일의 목적

이 파일은 `test1.md`나 `test2.md`와 같은 임시 파일을 대체하며, 린트 테스트가 성공하는지 확인하기 위해 생성되었습니다.

### 주요 내용

* 간결하고 명확한 문단 구성.
* 정확한 마크다운 문법 사용.
* 불필요한 공백 제거.
